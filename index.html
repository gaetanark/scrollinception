A way to avoid scroll inception

A scroll inception is that awkward moment when you scroll an embeded scrollable block and not the page.

Like that…

You don’t like… Here is a solution for an iframe

Or a scrollable div

How to use this

  1. Get jQuery, or use a CDN, like jsDelivr
  2. Get the jQuery plugin;
  3. Add these to your page;
  4. Any element using data-scrollinception="true" will use the plugin.
    (Ex. <iframe src="//lorem.com" … data-scrollinception="true"></iframe> )
